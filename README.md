# HomeAlarmSystem

##Features 
* Set A Passwoword
* ActivateAlarm
* DeactivateAlarm
* Alarm will be played if the Alarm is Activated and the door is opened and the correct password is not entered
* if alarm is decativated one can easily close and open the door without setting of an alarm
